# Thoughtful Interaction Design
Abstract-Pad handles complex interaction gracefully:

- Drawing is limited strictly to the canvas
- UI elements remain fully interactive
- Sliders are protected from canvas event hijacking
- Desktop and mobile behavior remain consistent

This project required careful coordination between p5.js canvas events and DOM UI controls, especially for touch devices.
